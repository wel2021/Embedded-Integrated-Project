### 3. 自动化脚本

#### 🚀 start.sh - 一键启动脚本（全新）
**功能**:
```bash
./start.sh setup      # 初始化环境（安装依赖、编译、配置数据库）
./start.sh start      # 启动所有服务
./start.sh stop       # 停止所有服务
./start.sh restart    # 重启所有服务
./start.sh status     # 查看服务状态
./start.sh help       # 显示帮助
```

**特性**:
- ✅ 彩色输出，易于阅读
- ✅ 自动依赖检查
- ✅ PID 文件管理进程
- ✅ 日志记录到文件
- ✅ 错误处理和提示
- ✅ 幂等性（可重复执行）

**技术实现**:
- Bash 脚本
- 进程管理
- 信号处理
- 日志轮转
